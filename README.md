# 银行渠道业务管理系统
> * 后端：java、springboot、mybatis
> * 前端：vue、element-ui、axios
> * 数据库：mysql


| 文件                              | 说明        |
|---------------------------------|-----------|
| controller                      | 接口层       |
| mapper                          | 数据库持久层    |
| pojo                            | 数据库实体层    |
| result                          | 返回结果类     |
| service                         | 业务逻辑层接口   |
| service/impl                    | 业务逻辑层接口实现 |
| util                            | 工具类       |
| BankDemoApplication             | 程序启动入口    |
| resource                        | 资源文件      |
| resource/mapper                 | sql映射文件   |
| resource/application.properties | 配置文件      |
| pom.xml                         | maven依赖   |
